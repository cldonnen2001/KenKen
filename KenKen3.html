KenKen’s rules are straightforward:
  1. Fill in each square cell in the puzzle with a number between 1 and 4.
  2. Use each number exactly once in each row and each column.
  3. The numbers in each “Cage” (indicated by the heavy lines) must combine, in any order, to produce the cage’s target number using the indicated math operation.
    Numbers may be repeated within a cage as long as rule 2 isn’t violated.
  4. Each puzzle can be solved completely using only logical deduction.
